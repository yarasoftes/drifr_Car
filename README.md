
# 🏎️ Arena Drift

Многопользовательская 3D-игра на браузере. Неоновая арена, машинки, сбор монет и нитро-ускорений. Игроки видят друг друга в реальном времени.

---

## 🎮 Геймплей

- Управляй машиной на неоновой арене
- Собирай жёлтые **монеты** (+10 очков)
- Собирай синие **нитро-канистры** (+50 очков + бонус)
- Нажми **N** для нитро-ускорения
- Сталкивайся с другими игроками
- Зарабатывай очки и поднимайся в таблице лидеров

---

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| W / ↑ | Газ |
| S / ↓ | Тормоз |
| A / ← | Поворот влево |
| D / → | Поворот вправо |
| N | Нитро-ускорение |

## 🚀 Установка и запуск

### Требования

- **Node.js** v16+
- **MongoDB** v7+
- **ngrok** (для игры по интернету)

### 1. Установка зависимостей

```bash
cd server
npm install
```

### 2. Запуск MongoDB

```bash
# Windows (PowerShell)
Start-Service -Name MongoDB

# macOS
brew services start mongodb-community

# Ubuntu
sudo systemctl start mongodb
```

### 3. Запуск сервера

```bash
cd server
npm start
```

Сервер запустится на `http://localhost:3000` и будет отдавать и API, и клиент.

### 4. Локальная игра

Открой браузер → `http://localhost:3000`

### 5. Игра по интернету (для друзей)

```bash
разные сыллки при подключение ngrok
```

Отправь другу ссылку `https://xxxx.ngrok-free.dev`

## ⚠️ Примечания

- **ngrok** в бесплатной версии меняет ссылку при перезапуске
- Пароли хранятся в открытом виде (для простоты). В продакшене используй `bcrypt`
- MongoDB должна быть запущена перед стартом сервера
- Для игры по интернету нужен ngrok или белый IP с пробросом портов
